Submit


Botão utilizado em formulários, para enviar os dados do formulário para um servidor para que sejam processados.

Forma geral:
<INPUT TYPE="submit" NAME="objectName" VALUE="label" [onClick="ação"]>

onde:
submit - Tipo de botão que será mostrado.

objectName - Nome do botão, definido pelo programador.

label - Texto que aparece na face do botão.

ação - Determina o que fazer quando ocorrer o evento onClick.

Propriedades:
name - Armazena o nome do botão, que foi definido no campo NAME.

value - Armazena o texto que aparece na face do botão, definido no campo VALUE.

Eventos:
onClick - Ocorre quando clicamos o mouse sobre o botão.

Exemplo:
No fim da página anterior, onde aparece um formulário em que você pode votar em minha página para Top10, existe um botão "enviar", que é um exemplo de botão Submit.
